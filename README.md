## Installation

Clone my-project with git clone SSH
```bash
  git clone git@github.com:fyou00/fyou00.github.io.git
  cd fyou00
```

Run jekyll serve
```bash
  jekyll serve -livereload
```
Lalu akses web yang telah dibuat di web browser dengan alamat http://localhost:4000
